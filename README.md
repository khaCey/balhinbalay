# BalhinBalay - Find Your Dream Home in Cebu

A React-based property listing website for finding homes in Cebu, Philippines.

## Features

- Browse properties for sale and rent
- Filter by property type and price range
- View detailed property information with image carousels
- Responsive design for mobile and desktop

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## Project Structure

```
src/
  components/     # React components
  data/          # Data files (listings, price ranges)
  utils/         # Helper functions
  App.js         # Main application component
  App.css        # Application styles
  index.js       # Entry point
```
